## Installation

### Composer

```console
$ composer require klimick/doctrine-php-mapping
```

### Enable psalm plugin (optional, but recommended)

```console
$ vendor/bin/psalm-plugin enable Klimick\\PsalmDoctrinePhpMapping\\Plugin
```
